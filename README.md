# RPG Marketplace Backend

## Project overview
This project is a production-ready Java backend for a singleplayer RPG with a shared marketplace.
Players own items, manage inventories, and trade items through a global market API.

## Purpose
The project is developed incrementally during 3rd semester as a backend portfolio,
focusing on real-world Java backend development practices.

## Current scope
- Basic Java + Maven project setup
- Initial domain structure
- No authentication or persistence yet

## Planned features
- REST API
- Database persistence (JPA / PostgreSQL)
- Marketplace buy/sell logic
- Validation and error handling
- JWT authentication & authorization
- Integration tests
- External API integration
- Deployment

## Tech stack
- Java
- Maven
- (Later) JPA / Hibernate
- (Later) PostgreSQL
- (Later) JWT
- (Later) Docker
